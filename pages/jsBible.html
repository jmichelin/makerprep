mathematical operators

          x = x + 12; /* is the same as: */ x += 12;
          x = x - 12; /* is the same as: */ x -= 12;
          x = x * 12; /* is the same as: */ x *= 12;
          x = x / 12; /* is the same as: */ x /= 12;

to increment/decrement a number/variable by 1:

if the `++` or `--` are before the variable then it adds/subtracts 1 BEFORE it is used

          x++;
          x--;
          ++x;
          --x;
