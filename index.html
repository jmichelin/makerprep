What was required?

  1. All body content should be wrapped in a div (you can use divs in other places too)
  2. Some header tags
  3. Some paragraphs
  4. A "horizontal rule" delimiter
  5. Some formatted text (i.e. <em>, <strong>, etc.)
  6. One or more ordered or unordered lists
  7. Plenty of images (on your computer or on the web)
  8. At least a few SECRET COMMENTS that visitors can only see if they view your source code!
  9. Multiple pages (think different sections on a website that you can navigate between using links)
